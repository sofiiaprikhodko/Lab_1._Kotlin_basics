`sealed class` (ізольований клас) дозволяє створити сувору ієрархію. Усі спадкоємці такого класу повинні бути оголошені в тому ж пакеті. Це робить використання `when` вичерпним (без необхідності гілки `else`).
Оголосіть `sealed class NetworkResult`. Створіть у ньому два вкладені data-класи, які його наслідують: `Success` (має `val data: String`) та `Error` (має `val errorMsg: String`).
