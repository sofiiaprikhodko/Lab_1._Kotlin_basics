Об'єднаємо класи, кастомні властивості та `init`!
Створіть звичайний клас `Worker` з властивостями `name` (String) та `hourlyRate` (Int).
1. У блоці `init` перевірте, що ставка `hourlyRate` більше 0, інакше викиньте `IllegalArgumentException`.
2. Додайте змінну властивість (var) `hoursWorked: Int`, яка ініціалізується нулем. Встановіть їй кастомний сетер: нове значення записується у `field` тільки якщо воно >= 0.
3. Додайте властивість `val salary: Int` з кастомним гетером, що повертає добуток `hourlyRate * hoursWorked`.
