У Kotlin первинний конструктор оголошується прямо в заголовку класу. Якщо ви додасте `val` або `var` перед параметрами конструктора, вони автоматично стануть властивостями класу.
Створіть клас `Student`, який має первинний конструктор із двома властивостями: `name` (типу `String`, незмінна `val`) та `age` (типу `Int`, змінна `var`).
