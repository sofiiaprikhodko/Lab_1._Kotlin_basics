Щоб змінити логіку присвоєння значення змінній `var`, можна написати кастомний сетер. Усередині сетера для доступу до реального значення в пам'яті використовується ключове слово `field` (Backing Field), інакше виникне нескінченна рекурсія.
Додайте кастомний сетер для властивості `balance`. Якщо нове значення (`value`) менше нуля, залишайте старе значення (нічого не робіть). Інакше — оновлюйте `field`.
