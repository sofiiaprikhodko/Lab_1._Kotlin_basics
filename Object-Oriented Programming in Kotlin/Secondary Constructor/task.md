Клас може мати вторинні конструктори, визначені за допомогою ключового слова `constructor`. Кожен вторинний конструктор зобов'язаний викликати первинний (через `this`).
Клас `Rectangle` має первинний конструктор, що приймає `width` і `height`. Створіть вторинний конструктор, який приймає лише один параметр `size: Int` (для створення квадрата) і делегує виклик первинному конструктору, передаючи `size` для обох сторін.
