Оператор `!!` (Not-null assertion) перетворює Nullable тип у Non-Nullable. Якщо змінна дорівнює `null`, програма викине `NullPointerException`. Використовуйте його лише тоді, коли впевнені на 100%, що значення не `null`.
Викличте властивість `.length` у рядка `text`, примусово вказавши компілятору, що він не `null`, за допомогою `!!`.
