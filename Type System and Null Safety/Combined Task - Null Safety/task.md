Об'єднаємо всі отримані знання в одну логічну задачу!
Напишіть функцію, яка приймає `String?` (ім'я користувача) і генерує привітання.
1. Якщо `name` дорівнює `null`, використайте значення за замовчуванням `"Guest"`.
2. Якщо ім'я не `null`, але воно порожнє (`.isEmpty()`), поверніть `"Guest"`. *Підказка: метод `.takeIf { it.isNotEmpty() }` дуже допоможе, або використовуйте `if-else`*.
3. Поверніть фінальний рядок: `"Welcome, [Name]!"`.

Використайте комбінацію безпечних операторів та рядкових шаблонів, розглянутих у попередньому розділі.
