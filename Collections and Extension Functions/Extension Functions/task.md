Функції-розширення дозволяють додавати нові методи до існуючих класів (навіть стандартних типу `String` чи `Int`) без їх успадкування. Синтаксис: `fun Тип.назваФункції()`. Усередині функції ключове слово `this` вказує на поточний об'єкт.
Напишіть функцію-розширення `removeSpaces()` для класу `String`, яка замінює всі пробіли на порожні рядки (використайте метод `.replace(" ", "")`).
