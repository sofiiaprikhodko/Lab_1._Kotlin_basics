Хорошою практикою в Kotlin є створення змінного списку всередині функції для обробки даних, але повернення його назовні як незмінного `List`, щоб уникнути небажаних модифікацій.
Змініть тип повернення функції з `MutableList<String>` на `List<String>`. Сама функція повертає `MutableList`, але завдяки поліморфізму він буде безпечно приведений до `List`.
