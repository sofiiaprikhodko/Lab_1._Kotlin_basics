Замість циклу `for` у Kotlin часто використовують лямбда-вирази та функцію `forEach`. Усередині лямбди поточний елемент колекції доступний під неявним ім'ям `it`.
Дано список імен. Використайте `names.forEach { ... }`, щоб додати до змінної `result` кожне ім'я та кому з пробілом (наприклад, `it + ", "`).
