Іменовані аргументи дозволяють пропустити параметри за замовчуванням усередині списку і передати значення лише для тих, які знаходяться далі.
Функція `createWindow` приймає ширину, висоту та заголовок. Викличте її, залишивши `width` та `height` за замовчуванням, але змінивши `title` на `"My App"` за допомогою іменованого аргумента.
