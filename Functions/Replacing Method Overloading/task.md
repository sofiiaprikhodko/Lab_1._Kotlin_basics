У Java для обчислення об'єму куба (1 параметр) і паралелепіпеда (3 параметри) довелося б писати два різних методи (перевантаження). У Kotlin достатньо однієї функції.
Напишіть функцію `volume`, яка приймає `length`. Параметри `width` та `height` повинні за замовчуванням дорівнювати `length`. Функція має повертати `length * width * height`.
