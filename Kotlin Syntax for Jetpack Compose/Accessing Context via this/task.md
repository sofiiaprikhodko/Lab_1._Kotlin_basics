# Явний `this` у Scope
Оскільки лямбда виконується "всередині" об'єкта `Scope`, ключове слово `this` вказує саме на цей контекстний об'єкт.

Іноді вам потрібно явно послатися на цей Scope, щоб передати його в іншу функцію, або щоб розрізнити змінні з однаковими іменами (наприклад, коли властивість Scope має таке ж ім'я, як зовнішня змінна).

**Ваше завдання:**
Функція `Column` надає контекст `ColumnScope`. У цьому класі є властивість `name`. Викличте `Column { ... }` і поверніть значення цієї властивості, явно використовуючи `this.name`.
