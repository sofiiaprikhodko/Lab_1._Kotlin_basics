# Комбінація синтаксису
Декларативні компоненти часто приймають обробники подій (наприклад, `onClick`) та контент, який вони повинні відображати. Щоб зробити код кристально чистим, розробники комбінують іменовані аргументи в круглих дужках та лямбду контенту за їх межами.

Замість того, щоб писати `Button("Submit", { "Text inside" })`, ми пишемо `Button(onClick = "Submit") { "Text inside" }`. Круглі дужки конфігурують компонент, а фігурні — наповнюють його.

**Ваше завдання:**
Викличте функцію `Button`. Використайте іменований аргумент `onClick = "Submit"` у круглих дужках. Фігурні дужки з лямбдою винесіть назовні, всередині неї поверніть рядок `"Text inside"`.
