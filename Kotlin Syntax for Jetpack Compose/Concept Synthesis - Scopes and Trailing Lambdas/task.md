# Динамічні списки та їхні Scope
У складних компонентах, таких як ліниві списки, лямбда з одержувачем (Scope) дозволяє декларативно оголошувати, які елементи мають бути всередині.

Функція `LazyColumn` надає контекст `LazyItemScope`, у якому є метод `item`. Цей метод, у свою чергу, також приймає лямбду з контентом. Звучить складно, але синтаксично виглядає як чистий і зрозумілий текст.

**Ваше завдання:**
Викличте `LazyColumn` без круглих дужок. Усередині лямбди ви автоматично отримаєте доступ до `LazyItemScope`. Викличте там метод `item` (також без круглих дужок), всередині лямбди якого поверніть рядок `"Data"`.
