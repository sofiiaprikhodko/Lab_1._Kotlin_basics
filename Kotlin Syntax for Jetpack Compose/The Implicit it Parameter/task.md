# Ключове слово `it`
У Java, створюючи лямбду з одним аргументом, ви зобов'язані придумати йому ім'я: `value -> System.out.println(value)`. Часто це призводить до створення коротких безглуздих назв типу `v`, `e`, `it`.

Kotlin вирішив автоматизувати цей процес. Якщо лямбда-вираз має рівно один параметр, ви можете взагалі його не оголошувати. Компілятор автоматично надасть доступ до цього значення через ключове слово `it` (вбудоване ім'я параметра). Це робить короткі обробники дуже лаконічними.

**Ваше завдання:**
Функція `ListItem` виконує передану лямбду, передаючи в неї число `42`. Викличте `ListItem` і всередині фігурних дужок просто поверніть форматований рядок `"Item $it"`. Не пишіть `x -> ...`.
