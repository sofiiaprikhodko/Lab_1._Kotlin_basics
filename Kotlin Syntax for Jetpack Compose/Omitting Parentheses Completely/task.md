# Опускання ()
Логічне продовження попереднього правила. Якщо функція приймає **тільки один аргумент** (і це лямбда), ми можемо винести її за дужки. Але тоді круглі дужки залишаються порожніми: `Column() { ... }`.

Щоб не засмічувати код, Kotlin дозволяє повністю видалити порожні круглі дужки. В результаті виклик функції стає таким: `Column { ... }`. Це виглядає як структура самої мови, створюючи елегантний Domain-Specific Language (DSL).

**Ваше завдання:**
Функція `Column` приймає лише один аргумент — лямбду `content`. Викличте її, не використовуючи круглі дужки взагалі. Усередині фігурних дужок поверніть рядок `"Item"`.
