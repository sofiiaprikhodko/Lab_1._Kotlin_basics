# Розширення для `Float`
Розширення можна писати для будь-яких типів. Для розмірів шрифту часто використовуються числа з плаваючою крапкою (наприклад, `24.5f.sp`). Для цього створюється аналогічне розширення, але вже для класу `Float`.

Це потужний інструмент, який робить код декларативним: ми не викликаємо утилітарний метод `DensityConverter.toSp(24.5f)`, а пишемо розмір так, як ми вимовляємо його вголос.

**Ваше завдання:**
Напишіть властивість-розширення `sp` типу `String` для класу `Float`. Геттер повинен повертати рядок `"${this}sp"`.
