# Безпека типів в UI
Завдяки різним Scope розробники фреймворків можуть гарантувати, що ви не застосуєте горизонтальне вирівнювання у вертикальному списку, адже такі методи просто не будуть існувати в поточному контексті.

- Метод `weight()` (заповнення доступної ширини) визначено тільки в `RowScope`.
- Метод `gravity()` визначено тільки в `ColumnScope`.

**Ваше завдання:**
Створіть рядок, об'єднавши результати двох викликів. Спочатку викличте `Row` і всередині нього поверніть виклик `weight()`. Потім додайте до результату (через оператор `+`) виклик `Column`, всередині якого поверніть `gravity()`.
