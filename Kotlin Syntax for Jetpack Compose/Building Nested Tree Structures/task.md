# Деревоподібний код без зайвого шуму
Коли ми прибираємо круглі дужки з компонентів, що є контейнерами для інших компонентів, ми отримуємо чисту ієрархію дерева. Це кардинально відрізняється від Java, де вкладеність вимагала б створення об'єктів та виклику методів `addView()`.

Завдяки синтаксису Kotlin структура коду повністю відповідає візуальній структурі UI: батьківський блок містить фігурні дужки, в яких лежать дочірні блоки.

**Ваше завдання:**
Створіть вкладену структуру. Викличте контейнер `Column` (без дужок). Всередині його лямбди викличте контейнер `Row` (без дужок). А вже всередині лямбди `Row` поверніть рядок `"Text"`.
