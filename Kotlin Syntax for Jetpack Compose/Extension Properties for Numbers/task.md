# Як працює "магія" одиниць виміру
Коли Java-розробник бачить код на кшталт `width = 16.dp`, перша думка — це вбудований синтаксис мови. Але це не так!

Kotlin дозволяє додавати не лише нові методи до існуючих класів (Extension Functions), але й нові **властивості** (Extension Properties). У даному випадку розробники просто додали властивість `dp` до стандартного класу цілих чисел `Int`. Значення самого числа доступне всередині гетера через ключове слово `this`.

**Ваше завдання:**
Оголосіть властивість-розширення `dp` типу `String` для класу `Int`. У її гетері (`get()`) реалізуйте повернення рядка формату `"[значення_числа]dp"` (наприклад, `"16dp"`). Використовуйте `this`.
