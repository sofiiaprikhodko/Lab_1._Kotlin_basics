# Підсумок: Створення UI дерева
Ви пройшли шлях від розуміння базових функцій до концепцій Scope та іменованих аргументів. Тепер поєднаємо усе разом, щоб написати фрагмент коду, який виглядає на 100% як реальний декларативний UI на Kotlin.

Ваш код перестав бути набором викликів об'єктів (як у Java) і став справжньою мовою розмітки (DSL), залишаючись при цьому суворо типізованим та безпечним на етапі компіляції.

**Ваше завдання:**
Побудуйте та поверніть таку ієрархію UI:
1. `Column` (як контейнер для всього, без круглих дужок). Усередині нього:
2. `Button`, де в круглих дужках вказано `onClick = "Save"`. А в його фігурних дужках (Trailing Lambda):
3. `Text` із переданим рядком `"Save"` у круглих дужках.
